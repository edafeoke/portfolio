**Affordable** is an eCommerce website that helps users find and purchase **fairly used computers** at affordable prices. The platform is built with **Next.js** for performance, **Tailwind CSS** for rapid UI development, and is optimized for SEO to drive organic discovery.

## Features

- **Product Listing & Search**: Clean and organized catalog of fairly used PCs with intuitive search.
- **Responsive Design**: Works seamlessly across desktop, tablet, and mobile.
- **Fast & SEO-Optimized**: Pages are pre-rendered for speed and discoverability.
- **User-Friendly Interface**: Simple and modern design focused on ease of use.

## Technologies

- **Next.js**: React framework for building high-performance web apps with static and dynamic routes.
- **Tailwind CSS**: Utility-first CSS framework for flexible and efficient styling.
- **TypeScript**: Used throughout the codebase for type safety.
- **Vercel**: Instant deployment platform with CI/CD integration.
- **SEO Optimization**: Structured metadata, sitemap, and social sharing tags.
